# Better Window for Ballance

这是一个用于 [Ballance](https://ballance.jxpxxzj.cn) 的 Mod，旨在让 Ballance 窗口有更加正常的体验。

## 依赖

- [BallanceModLoader](https://github.com/Gamepiaynmo/BallanceModLoader/releases)

## 功能列表

- 自动阻止窗口在后台获取键盘原始输入（方便将 Ballance 切至后台）
	- 此功能默认开启，可在设置中选择关闭
- 通过快捷键阻挡 Ballance 获取键盘输入
- `Alt + F4` 可快速退出游戏
